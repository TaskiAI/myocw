  1. Suppose we know that A is an m by n matrix of rank r, Ax=[100]Ax = \begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix}Ax=​100​​ has no solution, and Ax=[010]Ax = \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix}Ax=​010​​ has exactly one solution.

a) What can we say about m, n and r? The product Ax is a vector in three dimensions, so m=3m = 3m=3. The fact that Ax=[100]Ax = \begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix}Ax=​100​​ has no solution tells us that the column space is not all of R3R^3R3. In addition, we know that the column space contains [010]\begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix}​010​​, so r is not zero: 1<r<31 < r < 31<r<3. The fact that Ax=[010]Ax = \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix}Ax=​010​​ has exactly one solution tells us that the nullspace of A contains only the zero vector and so n=rn = rn=r. Hence 1<n<31 < n < 31<n<3.

b) Write down an example of a matrix A that fits this description. The vector [010]\begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix}​010​​ must be in the column space, so we'll make it a column of A. The simplest way to answer this question is to stop here.

A=[010]A = \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix}A=​010​​

In this solution, n=r=1n = r = 1n=r=1 and m=3m = 3m=3. To find a solution in which n=r=2n = r = 2n=r=2, add a second column. Make sure that [100]\begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix}​100​​ is not in the column space:

A=[001001]A = \begin{bmatrix} 0 & 0 \\ 1 & 0 \\ 0 & 1 \end{bmatrix}A=​010​001​​

There are many other correct answers to this question.

ATAA^T AATA is invertible iii. AATA A^TAAT is positive definite One good approach to this problem is to use our sample matrix to test each statement. i. If we leave this part to last, we can quickly answer it (false) using what we learn while answering the following two parts. ii. The matrix ATAA^T AATA is invertible if r=nr = nr=n; i.e. if the columns of A are independent. The nullspace of our A contains only the zero vector, so this statement is true. For each of our sample matrices, ATAA^T AATA equals the identity and so is invertible. Note that this means det⁡ATA≠0\det A^T A \ne 0detATA=0. iii. We know that m=3m = 3m=3 and r<3r < 3r<3, so AATA A^TAAT will be a 3 by 3 matrix with rank less than 3; it can't be positive definite. d) Prove that ATy=cA^T y = cATy=c has at least one solution for every right hand side c, and in fact has infinitely many solutions for every c. We know ATA^TAT is an n by m matrix with m=3m = 3m=3 and rank r=n<mr = n < mr=n<m. If ATA^TAT has full row rank, the equation ATy=cA^T y = cATy=c is always solvable. We have n rows and rank r=nr = nr=n, so ATA^TAT has full row rank. Therefore ATy=cA^T y = cATy=c has a solution for every vector c. The solvable system ATy=cA^T y = cATy=c will have infinitely many solutions if the nullspace of ATA^TAT has positive dimension. We know dim⁡(N(AT))=m−r>0\dim(N(A^T)) = m - r > 0dim(N(AT))=m−r>0, so ATy=cA^T y = cATy=c has infinitely many solutions for every c.
