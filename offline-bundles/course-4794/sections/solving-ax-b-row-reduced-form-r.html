Solving Ax=bAx = bAx=b: row reduced form RRR

When does Ax=bAx = bAx=b have solutions xxx, and how can we describe those solutions?

Solvability conditions on bbb

We again use the example:

A=(1222246836810)A = \begin{pmatrix} 1 & 2 & 2 & 2 \\ 2 & 4 & 6 & 8 \\ 3 & 6 & 8 & 10 \end{pmatrix}A=​123​246​268​2810​​

The third row of AAA is the sum of its first and second rows, so we know that if Ax=bAx = bAx=b the third component of bbb equals the sum of its first and second components. If bbb does not satisfy b3=b1+b2b_3 = b_1 + b_2b3​=b1​+b2​ the system has no solution. If a combination of the rows of AAA gives the zero row, then the same combination of the entries of bbb must equal zero.

One way to find out whether Ax=bAx = bAx=b is solvable is to use elimination on the augmented matrix. If a row of AAA is completely eliminated, so is the corresponding entry in bbb. In our example, row 3 of AAA is completely eliminated:

If Ax=bAx = bAx=b has a solution, then b3−b2−b1=0b_3 - b_2 - b_1 = 0b3​−b2​−b1​=0. For example, we could choose

From an earlier lecture, we know that Ax=bAx = bAx=b is solvable exactly when bbb is in the column space C(A)C(A)C(A). We have these two conditions on bbb; in fact they are equivalent.

Complete solution

In order to find all solutions to Ax=bAx = bAx=b we first check that the equation is solvable, then find a particular solution. We get the complete solution of the equation by adding the particular solution to all the vectors in the nullspace.

A particular solution

One way to find a particular solution to the equation Ax=bAx = bAx=b is to set all free variables to zero, then solve for the pivot variables.

For our example matrix A, we let x2=x4=0x_2 = x_4 = 0x2​=x4​=0 to get the system of equations:

x1+2x3=1x_1 + 2x_3 = 1x1​+2x3​=1

which has the solution x3=3/2x_3 = 3/2x3​=3/2, x1=−2x_1 = -2x1​=−2. Our particular solution is:

xp=(−203/20)x_p = \begin{pmatrix} -2 \\ 0 \\ 3/2 \\ 0 \end{pmatrix}xp​=​−203/20​​

Rank

The rank of a matrix equals the number of pivots of that matrix. If A is an m by n matrix of rank r, we know r≤mr \le mr≤m and r≤nr \le nr≤n.

Full column rank

If r=nr = nr=n, then from the previous lecture we know that the nullspace has dimension n−r=0n - r = 0n−r=0 and contains only the zero vector. There are no free variables or special solutions.

If Ax=bAx = bAx=b has a solution, it is unique; there is either 0 or 1 solution. Examples like this, in which the columns are independent, are common in applications.

We know r≤mr \le mr≤m, so if r=nr = nr=n the number of columns of the matrix is less than or equal to the number of rows. The row reduced echelon form of the matrix will look like R=(I0)R = \begin{pmatrix} I \\ 0 \end{pmatrix}R=(I0​). For any vector b in Rm\mathbb{R}^mRm that's not a linear combination of the columns of A, there is no solution to Ax=bAx = bAx=b.

Full row rank

If r=mr = mr=m, then the reduced matrix R=(IF)R = \begin{pmatrix} I & F \end{pmatrix}R=(I​F​) has no rows of zeros and so there are no requirements for the entries of b to satisfy. The equation Ax=bAx = bAx=b is solvable for every b. There are n−r=n−mn - r = n - mn−r=n−m free variables, so there are n−mn - mn−m special solutions to Ax=0Ax = 0Ax=0.

Full row and column rank

If r=m=nr = m = nr=m=n is the number of pivots of A, then A is an invertible square matrix and R is the identity matrix. The nullspace has dimension zero, and Ax=bAx = bAx=b has a unique solution for every b in Rm\mathbb{R}^mRm.

Problem 8.3: (3.4 #36.)

Suppose Ax=bAx = bAx=b and Cx=bCx = bCx=b have the same (complete) solutions for every bbb. Is it true that A=CA = CA=C?

Solution:

Yes. In order to check that A=CA = CA=C as matrices, it is enough to check that Ay=CyAy = CyAy=Cy for all vectors yyy of the correct size (or just for the standard basis vectors, since multiplication by them "picks out the columns"). So let yyy be any vector of the correct size, and set b=Ayb = Ayb=Ay. Then yyy is certainly a solution to Ax=bAx = bAx=b, and so by our hypothesis must also be a solution to Cx=bCx = bCx=b; in other words, Cy=b=AyCy = b = AyCy=b=Ay.
