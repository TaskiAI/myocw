Projection Matrices and Least Squares
Lecture
Session Overview

Linear regression is commonly used to fit a line to a collection of data. The method of least squares can be viewed as finding the projection of a vector. Linear algebra provides a powerful and efficient description of linear regression in terms of the matrix ATA.

Projection matrices and least squares

Projections

Last lecture, we learned that P=A(ATA)−1ATP = A(A^TA)^{-1}A^TP=A(ATA)−1AT is the matrix that projects a vector bbb onto the space spanned by the columns of AAA. If bbb is perpendicular to the column space, then it's in the left nullspace N(AT)N(A^T)N(AT) of AAA and Pb=0Pb = 0Pb=0. If bbb is in the column space then b=Axb = Axb=Ax for some xxx, and Pb=bPb = bPb=b.

A typical vector will have a component ppp in the column space and a component eee perpendicular to the column space (in the left nullspace); its projection is just the component in the column space. The matrix projecting bbb onto N(AT)N(A^T)N(AT) is I–PI – PI–P:

e=b−pe = b - pe=b−p
e=(I−P)be = (I - P)be=(I−P)b

Naturally, I–PI – PI–P has all the properties of a projection matrix.

Least squares

The process we're going to use is called linear regression; this technique is most useful if none of the data points are outliers. By "closest" line we mean one that minimizes the error represented by the distance from the points to the line. We measure that error by adding up the squares of these distances. The matrix ATAA^T AATA

We've been assuming that the matrix ATAA^T AATA is invertible. Is this justified? If AAA has independent columns, then ATAA^T AATA is invertible. To prove this we assume that ATAx=0A^T Ax = 0ATAx=0, then show that it must be true that x=0x = 0x=0:

ATAx=0A^T Ax = 0ATAx=0
xTATAx=xT0x^T A^T Ax = x^T 0xTATAx=xT0
(Ax)T(Ax)=0(Ax)^T (Ax) = 0(Ax)T(Ax)=0
Ax=0Ax = 0Ax=0

Since AAA has independent columns, Ax=0Ax = 0Ax=0 only when x=0x = 0x=0. As long as the columns of AAA are independent, we can use linear regression to find approximate solutions to unsolvable systems of linear equations. The columns of AAA are guaranteed to be independent if they are orthonormal, i.e. if they are perpendicular unit vectors like [10]\begin{bmatrix} 1 \\ 0 \end{bmatrix}[10​] and [01]\begin{bmatrix} 0 \\ 1 \end{bmatrix}[01​], or like [cos⁡θsin⁡θ]\begin{bmatrix} \cos \theta \\ \sin \theta \end{bmatrix}[cosθsinθ​] and [−sin⁡θcos⁡θ]\begin{bmatrix} -\sin \theta \\ \cos \theta \end{bmatrix}[−sinθcosθ​].
