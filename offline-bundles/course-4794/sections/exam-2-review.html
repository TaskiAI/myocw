Material covered by the exam

  • Orthogonal matrices Q=[q1…qn]Q = [ \mathbf{q}_1 \quad \dots \quad \mathbf{q}_n ]Q=[q1​…qn​]. QTQ=IQ^T Q = IQTQ=I.
  • Projections – Least Squares “best fit" solution to Ax=bAx = bAx=b.
  • Gram-Schmidt process for getting an orthonormal basis from any basis.
  • det⁡A\det AdetA
  • Properties 1-3 that define the determinant.
  • Big formula for the determinant with n!n!n! terms, each with +++ or −-−.
  • Cofactors formula, leading to a formula for A−1A^{-1}A−1.
  • Eigenvalues Ax=λxAx = \lambda xAx=λx.
  • det⁡(A−λI)=0\det(A - \lambda I) = 0det(A−λI)=0.
  • Diagonalization: If AAA has nnn independent eigenvectors, then S−1AS=ΛS^{-1}AS = \LambdaS−1AS=Λ (this is Ax=λxAx = \lambda xAx=λx for all nnn eigenvectors at once).
  • Powers of AAA: Ak=(SAS−1)k=SAkS−1A^k = (SAS^{-1})^k = SA^kS^{-1}Ak=(SAS−1)k=SAkS−1.
