Session Overview

We’d like to be able to “invert A” to solve Ax = b, but A may have only a left inverse or right inverse (or no inverse). This discussion of how and when matrices have inverses improves our understanding of the four fundamental subspaces and of many other key topics in the course.

Lecture Notes

Left and right inverses; pseudoinverse

Although pseudoinverses will not appear on the exam, this lecture will help us to prepare.

Two sided inverse

A 2-sided inverse of a matrix AAA is a matrix A−1A^{-1}A−1 for which AA−1=I=A−1AAA^{-1} = I = A^{-1}AAA−1=I=A−1A. This is what we've called the inverse of AAA. Here r=n=mr = n = mr=n=m; the matrix AAA has full rank.

Left inverse

Recall that AAA has full column rank if its columns are independent; i.e. if r=nr = nr=n. In this case the nullspace of AAA contains just the zero vector. The equation Ax=bAx = bAx=b either has exactly one solution xxx or is not solvable.

The matrix ATAA^T AATA is an invertible n×nn \times nn×n symmetric matrix, so (ATA)−1ATA=I(A^T A)^{-1}A^T A = I(ATA)−1ATA=I. We say Aleft−1=(ATA)−1ATA_{left}^{-1} = (A^T A)^{-1}A^TAleft−1​=(ATA)−1AT is a left inverse of AAA. (There may be other left inverses as well, but this is our favorite.) The fact that ATAA^T AATA is invertible when AAA has full column rank was central to our discussion of least squares.

Note that AAleft−1A A_{left}^{-1}AAleft−1​ is an m×mm \times mm×m matrix which only equals the identity if m=nm = nm=n. A rectangular matrix can't have a two sided inverse because either that matrix or its transpose has a nonzero nullspace.

Right inverse

If AAA has full row rank, then r=mr = mr=m. The nullspace of ATA^TAT contains only the zero vector; the rows of AAA are independent. The equation Ax=bAx = bAx=b always has at least one solution; the nullspace of AAA has dimension n−mn - mn−m, so there will be n−mn - mn−m free variables and (if n>mn > mn>m) infinitely many solutions!

Matrices with full row rank have right inverses Aright−1A_{right}^{-1}Aright−1​ with AAright−1=IA A_{right}^{-1} = IAAright−1​=I. The nicest one of these is AT(AAT)−1A^T (A A^T)^{-1}AT(AAT)−1. Check: AAA times AT(AAT)−1A^T (A A^T)^{-1}AT(AAT)−1 is III.

Pseudoinverse

An invertible matrix (r=m=nr = m = nr=m=n) has only the zero vector in its nullspace and left nullspace. A matrix with full column rank r=nr = nr=n has only the zero vector in its nullspace. A matrix with full row rank r=mr = mr=m has only the zero vector in its left nullspace. The remaining case to consider is a matrix AAA for which r<nr < nr<n and r<mr < mr<m.

If AAA has full column rank and Aleft−1=(ATA)−1ATA_{left}^{-1} = (A^T A)^{-1}A^TAleft−1​=(ATA)−1AT, then

AAleft−1=A(ATA)−1AT=PAA_{left}^{-1} = A(A^T A)^{-1}A^T = PAAleft−1​=A(ATA)−1AT=P

is the matrix which projects RmR^mRm onto the column space of AAA. This is as close as we can get to the product AM=IAM = IAM=I.

Similarly, if AAA has full row rank then Aright−1A=AT(AAT)−1AA_{right}^{-1}A = A^T (A A^T)^{-1}AAright−1​A=AT(AAT)−1A is the matrix which projects RnR^nRn onto the row space of AAA.

It's nontrivial nullspaces that cause trouble when we try to invert matrices. If Ax=0Ax = 0Ax=0 for some nonzero xxx, then there's no hope of finding a matrix A−1A^{-1}A−1 that will reverse this process to give A−10=xA^{-1}0 = xA−10=x.

The vector AxAxAx is always in the column space of AAA. In fact, the correspondence between vectors xxx in the (rrr dimensional) row space and vectors AxAxAx in the (rrr dimensional) column space is one-to-one. In other words, if x≠yx \neq yx=y are vectors in the row space of AAA then Ax≠AyAx \neq AyAx=Ay in the column space of AAA. (The proof of this would make a good exam question.)

Proof that if x≠yx \neq yx=y then Ax≠AyAx \neq AyAx=Ay

Suppose the statement is false. Then we can find x≠yx \neq yx=y in the row space of AAA for which Ax=AyAx = AyAx=Ay. But then A(x−y)=0A(x - y) = 0A(x−y)=0, so x−yx - yx−y is in the nullspace of AAA. But the row space of AAA is closed under linear combinations (like subtraction), so x−yx - yx−y is also in the row space. The only vector in both the nullspace and the row space is the zero vector, so x−y=0x - y = 0x−y=0. This contradicts our assumption that xxx and yyy are not equal to each other.

We conclude that the mapping x→Axx \rightarrow Axx→Ax from row space to column space is invertible. The inverse of this operation is called the pseudoinverse and is very useful to statisticians in their work with linear regression – they might not be able to guarantee that their matrices have full column rank r=nr = nr=n.

Finding the pseudoinverse A+A^+A+

The pseudoinverse A+A^+A+ of AAA is the matrix for which x=A+Axx = A^+Axx=A+Ax for all xxx in the row space of AAA. The nullspace of A+A^+A+ is the nullspace of ATA^TAT.

We start from the singular value decomposition A=UΣVTA = U \Sigma V^TA=UΣVT. Recall that Σ\SigmaΣ is an m×nm \times nm×n matrix whose entries are zero except for the singular values σ1,σ2,...,σr\sigma_1, \sigma_2, ..., \sigma_rσ1​,σ2​,...,σr​ which appear on the diagonal of its first rrr rows. The matrices UUU and VVV are orthonormal and therefore easy to invert. We only need to find a pseudoinverse for Σ\SigmaΣ.

The closest we can get to an inverse for Σ\SigmaΣ is an n×mn \times mn×m matrix Σ+\Sigma^+Σ+ whose first rrr rows have 1/σ1,1/σ2,...,1/σr1/\sigma_1, 1/\sigma_2, ..., 1/\sigma_r1/σ1​,1/σ2​,...,1/σr​ on the diagonal. If r=n=mr = n = mr=n=m then Σ+=Σ−1\Sigma^+ = \Sigma^{-1}Σ+=Σ−1. Always, the product of Σ\SigmaΣ and Σ+\Sigma^+Σ+ is a square matrix whose first rrr diagonal entries are 111 and whose other entries are 000.

If A=UΣVTA = U \Sigma V^TA=UΣVT then its pseudoinverse is A+=VΣ+UTA^+ = V \Sigma^+ U^TA+=VΣ+UT. (Recall that QT=Q−1Q^T = Q^{-1}QT=Q−1 for orthogonal matrices U,VU, VU,V or QQQ.)

We would get a similar result if we included non-zero entries in the lower right corner of Σ+\Sigma^+Σ+, but we prefer not to have extra non-zero entries.

Conclusion

Although pseudoinverses will not appear on the exam, many of the topics we covered while discussing them (the four subspaces, the SVD, orthogonal matrices) are likely to appear.
